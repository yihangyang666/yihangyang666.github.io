# 李墨飞扬的个人网站

这是一个使用 Next.js 和 Layui 构建的个人简历网站，展示了我的个人信息、教育背景、技能特长和工作经历。

## 🌟 特点

- 响应式设计，适配各种设备
- 优雅的动画效果
- 清晰的信息展示
- 现代化的UI界面
- 简洁的交互体验

## 🛠 技术栈

- **前端框架**: Next.js 14
- **UI组件**: Layui 2.9.23
- **样式处理**: TailwindCSS
- **开发语言**: TypeScript
- **部署平台**: GitHub Pages

## 📋 功能模块

1. **个人信息**
   - 基本信息展示
   - 联系方式
   - 教育背景

2. **主修课程**
   - 核心课程列表
   - 专业方向课程

3. **技能特长**
   - 专业证书
   - 获奖情况

4. **个人经历**
   - 校园经历
   - 竞赛经历
   - 项目经验

## 🚀 本地开发

1. 克隆项目
```bash
git clone https://github.com/yihangyang666/yihangyang666.github.io.git
cd yihangyang666.github.io
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 打开浏览器访问 `http://localhost:3000`

## 📦 部署

项目使用 GitHub Pages 部署，执行以下命令即可部署：

```bash
npm run deploy
```

## 🔧 自定义

1. 修改个人信息：编辑 `pages/index.tsx` 中的 `personalInfo` 对象
2. 更新照片：替换 `public/image/myphoto.jpg`
3. 更换背景：替换 `public/image/background.png`
4. 添加新的经历：编辑 `experiences` 数组
5. 更新技能：修改 `skills` 数组
