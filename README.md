# 李墨飞扬的个人网站

这是一个基于 Next.js 和 TailwindCSS 构建的个人网站项目。

## 功能特点

- 个人简介展示
- 教育背景和技能展示
- 项目经历展示
- 招聘推荐系统集成

## 技术栈

- Next.js 14
- TailwindCSS
- TypeScript
- React

## 本地开发

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 构建生产版本：
```bash
npm run build
```

4. 部署到 GitHub Pages：
```bash
npm run deploy
```

## 项目结构

```
.
├── components/       # React 组件
├── pages/           # 页面文件
├── public/          # 静态资源
├── styles/          # 样式文件
└── lib/            # 工具函数
```
